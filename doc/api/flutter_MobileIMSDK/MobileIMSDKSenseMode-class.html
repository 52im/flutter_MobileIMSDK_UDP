MobileIMSDKSenseMode enum

SenseMode3S 此模式下： KeepAlive心跳问隔为3秒； 10秒后未收到服务端心跳反馈即认为连接已断开（相当于连续3 个心跳间隔后仍未收到服务端反馈）。
